# Aula 1

## Projeto de Serviços

Entrar no site do [Spring Initializr](https://start.spring.io/) e gerar um novo projeto com as seguintes informações:

* **Project:** Maven
* **Language:** Java
* **Spring Boot:** 2.2.4
* **Project Metadata:**
  * **Group:** com.example
  * **Artifact:** aula1
* **Dependencies:**
  * Spring Web.
  * Spring Data JPA.
  * H2 Database.
